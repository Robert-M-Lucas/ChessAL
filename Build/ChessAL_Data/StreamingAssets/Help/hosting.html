Hosting

Basics

Click the 'Host' button and enter your name, the lobby's password (if you want it to have one) and select a gamemode or a save file. Once you have hosted a game you can click on a players team or their position in that team to change it. You will not be able to start a game until you have the correct number of players on the correct number of teams.

For players on the same network

To allow players on the same network (e.g. connected to the same Wi-Fi) to connect to your game you need to give them your local IP address (you can find this on screen once you've created the lobby). In the future this will be displayed to you.

For players across the internet

To allow players from anywhere to join you must give them your public IP address (you can find this on screen once you've created the lobby). You also need to forward port 8108.

Port forwarding

Every router (box that provides your internet) does this differently so there won't be any brand specific instructions. You can find information on how to forward a port by searching 'port forwarding [brand name]' e.g. 'port forwarding Virgin Media'. The typical process is navigating to an address in a web browser such as 192.168.0.1 or 192.168.1.1, entering your device's local IP address, setting the port in and the port out to 8108 and setting the mode to TCP or both.

Information you may need:

  • IP: [your local IP address]
  • Port: 8108
  • Mode: TCP or Both
Common errors
  • You don't see your local or public IP on the lobby screen. This is probably due to the server used to find your public IP being blocked
  • You get a 'Host failed' message without a direct reason. There is a cooldown between reusing ports for sockets so you may need to wait before being able to restart the server.
